# Civic Access Hub

Civic Access Hub is a web-based platform that provides immigrants and first-generation families with easy access to trusted civic resources, including voting, employment, education, and public services.

## 🌍 Overview:

Many newcomers struggle to navigate government systems and find reliable information about voting, employment, education, and public services. This project simplifies that process by organizing trusted civic resources on a clear, user-friendly website.

## 🚀 Features

* 🌐 Bilingual support (English / Spanish)
* 🌙 Dark mode toggle
* 🔍 Search functionality for resources
* 📄 Printable resource guides
* 📱 Responsive design (mobile-friendly)

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript

## 🎯 Purpose

The goal of this project is to reduce confusion around civic systems and provide accessible, reliable information that helps individuals participate more fully in their communities.
